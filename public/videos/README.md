# Hero Carousel Videos

Add your MP4 video files here for the hero carousel background videos.

## Required Files

- `hero-bg-1.mp4` - First slide background video (8-10 seconds, 1280x720)
- `hero-bg-2.mp4` - Second slide background video (8-10 seconds, 1280x720)
- `hero-bg-3.mp4` - Third slide background video (8-10 seconds, 1280x720)

## Specifications

- **Format**: MP4 (H.264 codec)
- **Duration**: 8-10 seconds (will loop)
- **Resolution**: Minimum 1280x720, recommended 1920x1080
- **File Size**: 2-5 MB per video
- **Codec**: H.264 video, AAC audio
- **Frame Rate**: 24-30 fps

## Notes

- Videos will autoplay (muted) and loop seamlessly
- If a video fails to load, the carousel will fallback to the static image
- See `HERO_MEDIA_SETUP.md` in the project root for detailed setup instructions
