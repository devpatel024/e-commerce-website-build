# Hero Carousel Ambient Audio

Add your audio file here for the hero carousel ambient soundtrack.

## Required File

- `ambient-luxury.mp3` - Ambient background music/soundtrack

## Specifications

- **Format**: MP3 or WAV
- **Duration**: 30-60 seconds (will loop seamlessly)
- **Bitrate**: 128 kbps MP3 or equivalent
- **Volume**: Should be soft and ambient (-20dB to -15dB mixing level)
- **No speech**: Instrumental or ambient nature sounds recommended

## Behavior

- Users can toggle audio on/off with the speaker button in the hero carousel
- Audio plays at 20% volume for subtle background ambiance
- User's audio preference is saved in localStorage
- Audio respects browser autoplay policies

## Notes

- Audio is completely optional - the carousel works perfectly without it
- See `HERO_MEDIA_SETUP.md` in the project root for detailed setup instructions
